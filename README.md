# DayLog

「理想」と「実際」を左右に並べて比較できる、スマートフォンに最適化されたスケジュール管理アプリです。

1日の予定（理想）と実際に過ごした時間（実際）を5分刻みで記録し、予定の管理や振り返りに活用できます。

（※PCでも使えないことはないようにしてありますが、スマホでしかできない操作が多々あります）

## 主な機能

- **1日ビュー**: 理想と実際を左右に並べて表示・編集できるメインの機能です

  「理想」欄に置いたブロックは編集画面から「実際欄にコピー」をクリックする、あるいは直接「実際」欄にドラッグすることで「実際」欄に転記できます
- **3日ビュー / 週間ビュー**: 左上の「1日」のボタンを押すことで、複数日を俯瞰して確認できる3日、1週間ビューに変更できます
- **検索**: 画面上部の虫眼鏡アイコンから、スケジュールブロックをタイトルで検索できます
- **TODOメモ**: ヘッダーのカレンダーマークから開くことができ、月間および年間のTODOなどのメモが記入できます。
- **理想の履歴**: 「理想」の文字の右にある時計アイコンから、理想欄のスケジュールのバージョン履歴が確認できます。刻一刻変化し崩れていった理想を比較・追跡できます（最新以外のいらない履歴は削除可能）
- **タイムゾーン対応**: 右上の設定アイコンから、異なるタイムゾーンに設定することで旅行先でも正しく表示できます
- **データのインポート / エクスポート**: 右上の設定アイコンから、JSON形式でデータのバックアップ・復元ができます
- **PWA対応**: ホーム画面に追加することでネイティブアプリのように使用できます

## ホーム画面への追加方法（スマホ）

1. Safari（iOS）または Chrome（Android）でアプリのURLを開く
2. 「ホーム画面に追加」でインストール
3. ホーム画面のアイコンからアプリを起動

## プライバシー

すべてのデータはデバイス内（localStorage）にのみ保存されます。外部サーバーへのデータ送信は一切行われません。
